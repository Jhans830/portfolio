IT Cosmetics Email Builder
  • About

    This tool was designed to empower email marketers at IT Cosmetics to generate custom template-based emails utilizing a simple user interface. To begin creating your email, simply click on the section of the email to the right that you wish to edit and then start adding your content. Please note that some sections of content are locked to ensure the integrity of the email layout and ease-of-use.

    For any questions or requests please reach out to your Jason Hanshaw contact about the tool.

  • Subject Line

    When creating a subject line for the email, there are a few rules to follow in order to ensure compatibility among all email clients.

    Length

    Among the most common clients, the amount of characters falls withing a fairly wide range (30 - 73 characters). Due to this, it's important to keep your subject lines short and direct to allow for maximum compatibility

    Special Characters

    Avoid using special characters within your subject lines unless absolutely necessary. Characters such as * % & # and ^ have been known to trigger spam filters.

    Emojis

    Using an emoji in your email subject line is as simple as copying the symbol or emoji from a website/document like Facebook Symbols, or Emojipedia, and pasting it into the subject line field of your email. Emojis enjoy almost universal support amongst email clients and, with the exception of Outlook 2003, will display as intented for all your customers.

  • Text

    There are several options that you can use to style and format your text within the email. Listed below is a glossary of the relevant functions you may use:

    Fonts Types and Sizes



    Both font-family and type can be chosen by selecting the appropriate value from the "font" and "size" drop down menus. The font types supported by this tool are the following: Arial, Comic Sans, Courier New, Georgia, Lucinda Sans Unicode, Tahoma, Times New Roman, Trebuchet, Verdana.

    Fonts Styles



    The following font-styles are supported by this tool: bold, italic, underline, strikethrough, subscript, superscripts, color and background-color.

    Text Alignment



    These icons can be used to set the alignment on a selection of text. Alignment can be left, center, right or justified.

    Lists and Indentations



    These icons allow you to create both ordered and bulleted lists, as well as providing the option to indent a selection of text.

  • Links

    This tool allows both text and images to be hyperlinked in order to direct the subscriber to external locations. For information on how to link an image within the email builder, please reference the image section of this navigation menu.

    Linking A Portion of Text



    In order to add a hyperlink to a portion of text, simply highlight the word or phrase that you wish to link and click on the anchor button displayed above. This should trigger the hyperlink menu to appear.

    Link Info



    In the Link Info dialog menu, there are several options that can be modified to deliver the desired hyperlink. You should only utilize the options listed below to ensure the functionality within the email will be rendered across clients:

    Link Type: Only URL and Email are valid options for this drop-down. If Email is selected, an additional menu will appear that will allow you to set the email address, subject line and body content for the mailto hyperlink.

    Protocol: http:// and https:// are the only valid protocols that you should select from this menu.

    URL: Use this input field to enter the desired location for your hyperlink to direct to.

    Link Target



    For the Link Target menu, the only valid input is "New Window (_blank)". This selection will ensure that your hyper link will appear in a new window.

    Link Advanced



    The Link Advanced menu allows for further styling of links outside of the primary editor. Only the following items should be considered when editing items in this menu.

    ID/Stylesheet Classes: Allows you to apply CSS styles that already exist within the email by specifying the name. Unfortunately, due to Outlook's inability to deviate from inline-styling, this option is very limited and will not be applied to all email clients.

    Style: This option allows you to write your own CSS styling to this link. Unlike the ID or Stylesheet Classes input, these styles will be inlined with the hyperlink and will therefore be applied in all email clients. It is not recommended to use this option unless you have a solid foundation of CSS knowledge.

  • Images

    This section outlines how to add images to the email using the build tool. It should be noted that the actual rendering of images within this tool may differ from what is presented in the actual email client. It is important to reference your actual test email or use tools such as Email on Acid to confirm image rendering across all clients.

    Adding An Image



    To add an image to your email, simply click on the image icon (shown above) to trigger the image menu within the builder.

    Image Info



    In the Image Info dialog menu, there are several options that can be modified to deliver the desired image output. You should only utilize the options listed below to ensure the functionality within the email will be rendered across clients:

    URL: Specify the URL to the image that you want to populate within the email.

    Alternative Text: Input the text that you want to appear in the event the user has not downloaded the image.

    Width: Specify the width of the image. If the image needs to extend the full width of the email then set this value to 100%, otherwise enter the exact width of the image (not to exceed 750px).

    Height: This value should always be left blank in order to ensure the image height will be auto-calculated.

    All other options should be left empty.

    Image Link



    In this menu, you can specify a hyperlink you would like to attach to the image. Note that the value for the "Target" property should always be set to "New Window (_blank)." For more information please reference the Link section of this navigation menu.

    Image Advanced



    The advanced image menu allows you to specify styling that you wish to apply to the image. It is recommended that these fields should remain blank unless you possess knowledge of CSS. For more information, please reference the Link section of this navigation menu.

  • Testing

    Spell Checking



    To enable spell checking for your email content, click on the section of text you wish to check and then select the icon shown above. After clicking this icon, select "Enable SCAYT" from the dropdown menu to enable spell checking on your content. If you currently use extensions, such as Grammarly, then those functions should perform checks automatically.

    Sending A Test Email

    Once you have finished editing the email and you wish to send a test, simply click the "Build Email" button at the bottom of the page and your email will be created and a test will be sent to a predefined list.

    If you have an question about the tool or wish to be added to the testing distribution list, then please reach out to Jason Hanshaw for assistance..
